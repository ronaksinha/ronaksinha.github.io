What is this website all about

The main reason I made this website was really to teach myself how to actually do it. I started off with a tutorial video on Pluralsight which taught the basics of front end development. I was able to recreate what the tutorial had shown but wanted to do more. I then created a website (an older version of this) which was very basic and similar to the structure of what the tutorial had shown; it wasn't personalized. Then, I set out to make what you're visiting today. This website was more of a personal challenge to push me to build something that is more my taste and to force me to learn more about front end development.

This website was made entirely from scratch using HTML, CSS, and JavaScript. A couple of things that I did not write are an API for the contact form and an external JS script to enable the scroll into and out animations. The source code lives of GitHub in a private repository. Yes, this is hosted using GitHub Pages.

This website is still a work in progress so there are some issues I'm aware of, but some that I am not. Do let me know if you come across some weird bugs or (more likely) responsiveness issues.
